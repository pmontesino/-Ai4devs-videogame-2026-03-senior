# Protocolo de Playtest

## Objetivo

Medir SC-001, SC-003, SC-004 y SC-009 con al menos 10 participantes internos.

## Preparacion

1. Reiniciar estado de guardado (cookie) antes de cada participante.
2. Usar una sesion por participante para evitar sesgo de aprendizaje.
3. Registrar navegador/version y dispositivo de prueba.
4. Mantener mismo guion de instrucciones para todos.

## Pasos

1. Arrancar partida nueva y registrar tiempo hasta primer control exitoso.
2. Pedir al jugador describir objetivo de mision antes de 3 minutos.
3. Medir completitud del primer tramo sin bloqueo critico.
4. Pedir uso de F1 y verificar entendimiento de controles.

## Reglas de scoring

- SC-001: PASS si >= 90% completa tramo 1 sin bloqueo critico.
- SC-003: PASS si >= 80% identifica objetivo antes de 3 min.
- SC-004: PASS si >= 85% domina control base en <= 60 s.
- SC-009: PASS si >= 90% comprende controles tras F1.

## Datos

- ID jugador
- Navegador
- Version
- Tiempo a control basico
- Comprension del objetivo
- Completa tramo 1
- Comprension de controles tras F1
- Observaciones
